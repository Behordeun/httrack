This means that we have implementations of all major features that we want to include in the 1.0 release. This release of EHRbase provides you with a nearly complete implementation of the official openEHR REST API.

Through the newly added Spring Transactions, EHRbase now guarantees coherent and reliable data handling from endpoint the database.

You will also find lots of fixes in the implementation of the Archetype Query Language, which enables the flexible and efficient access of your clinical data.

You can find more details about new features and bug fixes on the release notes: https://ehrbase.readthedocs.io/en/latest/01_release_notes/0100.html

Download the release of EHRbase at: https://github.com/ehrbase/ehrbase/releases/tag/v0.10.0
